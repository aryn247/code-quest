Tutorial

Welcome to CodeLearn! In this platform, you'll face various coding challenges that help you learn basic programming concepts. Here's how to play:

  • Commands: Type commands in the input box and press Enter.
  • Start: Type 'start' to begin the challenges.
  • Hints: If you're stuck, type 'hint' to get a clue (costs points).
  • Skip: You can skip up to 3 challenges by clicking the 'Skip Challenge' button (costs points).
  • Restart: Type 'restart' to start over.

Good luck, Coder!
